### 题目 1: 基础不定积分组合
计算：
$$\int \left( -3\sin(t) + \cos(t) + 4\sec^{2}(t) - 7e^{t} + \frac{7}{\sqrt{1-t^{2}}} + \frac{1}{1+t^{2}} \right) dt$$

**解析**：
这是一道考察基本积分公式的“送分题”。我们需要逐项积分。
1. $\int -3\sin(t) dt = 3\cos(t)$ （注意符号，$\cos$ 的导数是 $-\sin$）
2. $\int \cos(t) dt = \sin(t)$
3. $\int 4\sec^2(t) dt = 4\tan(t)$
4. $\int -7e^t dt = -7e^t$
5. $\int \frac{7}{\sqrt{1-t^2}} dt = 7\arcsin(t)$ （这是反正弦函数的导数形式）
6. $\int \frac{1}{1+t^2} dt = \arctan(t)$ （这是反正切函数的导数形式）

**最终答案**：
$$3\cos(t) + \sin(t) + 4\tan(t) - 7e^t + 7\arcsin(t) + \arctan(t) + C$$

### 题目 2: 有理函数积分 (假分式处理)
计算：
$$\int \frac{x^{2}+11x+5}{x+3} dx \quad (x+3>0)$$

**解析**：
被积函数是假分式（分子次数 $\ge$ 分母次数），解题第一步必须是**多项式除法**（或恒等变形），将其化为整式与真分式之和。

**步骤**：
1. **变形分子**：凑 $x+3$ 的项。
   $$x^2 + 11x + 5 = x(x+3) + 8x + 5 = x(x+3) + 8(x+3) - 24 + 5$$
   $$= x(x+3) + 8(x+3) - 19$$
2. **拆分分数**：
   $$\frac{x(x+3) + 8(x+3) - 19}{x+3} = x + 8 - \frac{19}{x+3}$$
3. **逐项积分**：
   $$\int (x + 8 - \frac{19}{x+3}) dx = \frac{1}{2}x^2 + 8x - 19\ln|x+3| + C$$
   *注：题目已知 $x+3>0$，所以绝对值符号可以去掉。*

**最终答案**：
$$\frac{1}{2}x^2 + 8x - 19\ln(x+3) + C$$

### 题目 3: 凑微分法 (换元)
计算：
$$\int \frac{x}{\sqrt{1-x^{4}}} dx$$

**解析**：
观察到分母中有 $x^4 = (x^2)^2$，而分子是 $x$（即 $x^2$ 的导数的一半），因此使用凑微分法。

**步骤**：
1. 令 $u = x^2$，则 $du = 2x dx \implies x dx = \frac{1}{2} du$。
2. 代入积分：
   $$\int \frac{1}{\sqrt{1-u^2}} \cdot \frac{1}{2} du = \frac{1}{2} \int \frac{1}{\sqrt{1-u^2}} du$$
3. 套用公式 $\int \frac{1}{\sqrt{1-x^2}} dx = \arcsin x$：
   $$= \frac{1}{2} \arcsin(u) + C$$
4. 回代 $u=x^2$。

**最终答案**：
$$\frac{1}{2}\arcsin(x^2) + C$$

### 题目 4: 指数函数积分
计算：
$$\int 6^{x}e^{x} dx$$

**解析**：
被积函数包含两个指数部分，我们可以利用指数运算法则合并底数。
$$6^x e^x = (6e)^x$$
利用指数积分公式 $\int a^x dx = \frac{a^x}{\ln a} + C$：

**最终答案**：
$$\frac{(6e)^x}{\ln(6e)} + C \quad \text{或者} \quad \frac{6^x e^x}{\ln 6 + 1} + C$$

### 题目 5: 分部积分法
计算：
$$\int 4x \ln(3x) dx$$

**解析**：
遇到“对数 $\times$ 幂函数”，首选**分部积分法**。根据 "LIATE" 原则（对数 Logarithmic 优先于代数 Algebraic），我们设对数为 $u$。

**步骤**：
1. 设 $u = \ln(3x)$， $dv = 4x dx$。
2. 微分 $u$：$du = \frac{1}{3x} \cdot 3 dx = \frac{1}{x} dx$。
3. 积分 $v$：$v = \int 4x dx = 2x^2$。
4. 套用公式 $\int u dv = uv - \int v du$：
   $$I = 2x^2 \ln(3x) - \int 2x^2 \cdot \frac{1}{x} dx$$
   $$I = 2x^2 \ln(3x) - \int 2x dx$$
   $$I = 2x^2 \ln(3x) - x^2 + C$$

**最终答案**：
$$2x^2 \ln(3x) - x^2 + C$$

### 题目 6: 定积分与分部积分
已知 $g(1)=-4, g(5)=3, \int_{1}^{5}g(x)dx=-6$，计算：
$$\int_{1}^{5} xg^{\prime}(x) dx$$

**解析**：
看到 $xg'(x)$ 就要想到分部积分，以此转移导数。

**步骤**：
1. 设 $u = x, dv = g'(x)dx$。
2. 则 $du = dx, v = g(x)$。
3. 应用定积分分部积分公式：
   $$\int_{1}^{5} xg^{\prime}(x) dx = [x g(x)]_{1}^{5} - \int_{1}^{5} g(x) dx$$
4. 代入数值：
   * 第一部分：$5g(5) - 1g(1) = 5(3) - 1(-4) = 15 + 4 = 19$
   * 第二部分：题目已知 $\int_{1}^{5} g(x) dx = -6$
5. 计算：$19 - (-6) = 25$

**最终答案**：
$$25$$

### 题目 7: 定积分换元 (第一类)
计算：
$$\int_{1}^{3} 24(x^{2}-6)^{5} x dx$$

**解析**：
观察到括号外有 $x$，正好是括号内 $x^2$ 导数的一部分。

**步骤**：
1. 设 $u = x^2 - 6$，则 $du = 2x dx \implies x dx = \frac{1}{2} du$。
2. **换限**（非常重要）：
   * 当 $x=1$ 时，$u = 1^2 - 6 = -5$。
   * 当 $x=3$ 时，$u = 3^2 - 6 = 3$。
3. 代入积分：
   $$\int_{-5}^{3} 24 \cdot u^5 \cdot \frac{1}{2} du = 12 \int_{-5}^{3} u^5 du$$
4. 计算：
   $$12 \cdot [\frac{u^6}{6}]_{-5}^{3} = 2 [u^6]_{-5}^{3} = 2(3^6 - (-5)^6)$$
   $$2(729 - 15625) = 2(-14896) = -29792$$

**最终答案**：
$$-29792$$

### 题目 8: 定积分换元 (第二类)
计算：
$$\int_{0}^{2} (e^{5x}-5x)^{5} (e^{5x}-1) dx$$

**解析**：
这题看起来很复杂，但仔细看后面一项 $(e^{5x}-1)$ 几乎就是底数的导数。
底数 $f(x) = e^{5x} - 5x$，其导数 $f'(x) = 5e^{5x} - 5 = 5(e^{5x}-1)$。

**步骤**：
1. 设 $u = e^{5x} - 5x$，则 $du = 5(e^{5x}-1) dx \implies (e^{5x}-1)dx = \frac{1}{5} du$。
2. **换限**：
   * 当 $x=0$ 时，$u = e^0 - 0 = 1$。
   * 当 $x=2$ 时，$u = e^{10} - 10$。
3. 积分：
   $$\int_{1}^{e^{10}-10} u^5 \cdot \frac{1}{5} du = \frac{1}{5} [\frac{u^6}{6}]_{1}^{e^{10}-10}$$
   $$= \frac{1}{30} [ (e^{10}-10)^6 - 1^6 ]$$

**最终答案**：
$$\frac{1}{30}((e^{10}-10)^6 - 1)$$

### 题目 9: 三角函数积分技巧

**(1)** $\int \frac{dx}{1+\cos 2x}$
**解析**：利用二倍角公式 $1+\cos 2x = 2\cos^2 x$。
$$\int \frac{1}{2\cos^2 x} dx = \frac{1}{2} \int \sec^2 x dx = \frac{1}{2} \tan x + C$$
**答案**：$0.5\tan(x) + C$

**(2)** $\int \frac{\cos 2x}{\cos^{2}x \sin^{2}x} dx$
**解析**：利用 $\cos 2x = \cos^2 x - \sin^2 x$ 拆分分子。
$$\int \frac{\cos^2 x - \sin^2 x}{\cos^2 x \sin^2 x} dx = \int (\frac{\cos^2 x}{\cos^2 x \sin^2 x} - \frac{\sin^2 x}{\cos^2 x \sin^2 x}) dx$$
$$= \int (\frac{1}{\sin^2 x} - \frac{1}{\cos^2 x}) dx = \int (\csc^2 x - \sec^2 x) dx$$
$$= -\cot x - \tan x + C$$
**答案**：$-\cot(x) - \tan(x) + C$

### 题目 10: 绝对值定积分
计算：
$$\int_{-2}^{2} -1|x^{2}-4x| dx$$

**解析**：
处理绝对值积分的关键是**去绝对值**。
1. **分析符号**：令 $x^2-4x = x(x-4) = 0$，零点为 $0, 4$。
2. **确定区间正负**：我们在 $[-2, 2]$ 区间内积分。
   * 当 $-2 \le x \le 0$ 时，$x$ 为负，$x-4$ 为负，乘积 $x(x-4)$ 为正。此时 $|x^2-4x| = x^2-4x$。
   * 当 $0 < x \le 2$ 时，$x$ 为正，$x-4$ 为负，乘积 $x(x-4)$ 为负。此时 $|x^2-4x| = -(x^2-4x) = 4x - x^2$。

3. **分段计算**：
   原式 $= - \int_{-2}^{2} |x^2-4x| dx$
   
   **第一段 $[-2, 0]$**:
   $$- \int_{-2}^{0} (x^2-4x) dx = - [\frac{x^3}{3} - 2x^2]_{-2}^{0}$$
   $$= - [0 - (\frac{-8}{3} - 2(4))] = - [0 - (-\frac{8}{3} - 8)] = - [\frac{32}{3}] = -\frac{32}{3}$$
   
   **第二段 $[0, 2]$**:
   $$- \int_{0}^{2} -(x^2-4x) dx = \int_{0}^{2} (x^2-4x) dx$$
   $$= [\frac{x^3}{3} - 2x^2]_{0}^{2} = (\frac{8}{3} - 8) - 0 = \frac{8-24}{3} = -\frac{16}{3}$$

4. **求和**：
   $$-\frac{32}{3} + (-\frac{16}{3}) = -\frac{48}{3} = -16$$

**最终答案**：
$$-16$$
