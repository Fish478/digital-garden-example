### 考古发现与中国文化


- 课程信息：

1. 授课老师：郭怡 / 蒋璐 / 林留根 / 考古专家

2. 课程学分：2.5学分

3. 分数构成：线上mooc 70%（作业+互评+自评），课程论文30%

4. 上课时间：周一11-13节，部分周不上课，部分周为专家讲座，可能有考古现场教学

5. 个人分数：93 / 4.8

  

- 推荐指数：9.0

  

- 课程优点：

1. 分数构成合理：70%的mooc部分，可以直接使用插件速通，耗时少。30%的论文部分，老师第一节课就表示会给认真写的同学高分。

2. 不存在点名：本学期没有点名

3. 课时少：看似是每周三节，但是本学期有近一个月没有上课

4. 老师好：林留根老师是我国大运河保护专家，上课的时候会给我们讲他参与文物保护斗争的故事，是一位有情怀有水平的老师；郭老师讲课幽默风趣，是一位很有趣的老师。

5. 可以聆听专家讲座，有机会参与前往考古发掘现场学习

  

- 课程缺点：mooc部分存在互评，互评就有一定的概率会存在恶意评分现象（）

  

- 学习经验：

1. mooc部分可以使用插件速通，亲自做的话确实比较耗时耗力

2. 论文部分一定要认真完成。老师虽然说了会尽量给高分，但是由于通识课有正态分布的强制要求，论文过于水或者AI味道太严重的话很可能被打低分。我写完之后给老师修改了一次，最后论文部分应该给分是给满了。

3. 如果平时比较忙的话，这门课全过程可以只花一个写论文和速通mooc的时间，不是非常需要记笔记之类的。

### 心理学导论

  

- 课程信息：

1. 授课老师：蔡永春 / 陈珂

2. 课程学分：2.0

3. 分数构成：平时成绩50%（小测 + 考勤 + 期中论文） + 期末考试50%(开卷) + 参与实验加分（上限五分，1分/小时）

4. 上课时间：周四11,12节

5. 个人分数：95 / 5.0

  

- 推荐指数：8.5

  

- 课程优点：

1. 考试为开卷，题量适中，难度适中

2. 可以学习到很多有趣的心理学知识，本课程是“心理学及应用”的升级版

3. 课后不需要花费额外时间

4. 给分与努力程度成正比

  

- 课程缺点：

1. 无法旷课，老师会不定时突击小测

2. 课堂没法彻底摸鱼（）突击小测内容往往和本节课所学高度相关

3. 论文格式要求较高，需要参考顶刊格式，引用英文文献可以加分

4. 期末考试部分题目似乎不是特别严谨，也有个别题目似乎翻不到

  

- 学习经验：

1. 考试部分：建议在考试前研究一下当年度的“心理学及应用”cyc班级的期末试卷，我们今年心导的前几题都是“心理学及应用”的原题搬过来的。

2. 考试补天：开卷考试的时候携带老师的ppt即可，几乎所有内容都在ppt中。平时也有涉及到一些比较难的知识点，这些知识点可能需要我们在期末复习的时候提前学习一下。我主要是借助AI重点复习了一下蔡永春老师授课的部分。有些题目还是考察理解的，如果一点不理解的话就不一定能写出来。

3. 小测建议：小测计入平时分，也同时相当于点名，所以心理学导论还是最好不要翘课好（）我们当时小测的题目是：阐述吊桥效应及其原理，还有另外一次是十多个选择题。想要小测获得高分课堂还是需要认真听讲的。

### 物理学与人类文明

  

- 课程信息：

1. 授课老师：叶高翔 / 盛正卯

2. 课程学分：2.0

3. 分数构成：课程论文 + 期末考试（闭卷）+ 小组展示（不太清楚算不算分）

4. 上课时间：周三11,12节

5. 个人得分：96 / 5.0

  

- 推荐指数：9.0

  

- 课程优点：

1. 叶老师和盛老师都是两位很有人文情怀的理科老师，真正的学者风范，先生之风上高水长！两位老师也很负责，每次咨询老师问题都能得到非常快、非常认真的回复，老师也很关心学生

2. 虽然是一门物理学的科普课程，但是正如这门课程的名字，很大一部分课程内容是和“人类文明”息息相关的，从这门课程中可以感受到两位老师的高远的科学思维和严谨的治学理念。物理学知识部分也深入浅出。

3. 总之是一门好课！文科生理科生都可以大胆选之！

  

- 学习经验：

1. 课堂：不听和听也许对期末考试影响不大（？）前期叶老师课堂基本还是认真听的，主要是知识点留了一个映像+感受叶老师的思想，后半学期实在太忙了，只能上课干点别的了o(╥﹏╥)o

2. 课程论文：可以尝试写出来之后给老师修改一下，论文我咨询了老师了两次，最后老师评价“已经是一个质量很高的论文了”(*╹▽╹*)

3. 课后整理：大部分时间花费在课程整理上了~

4. 考试补天：考试和历年卷的重合度还是比较高的。我推荐基于老师的ppt进行知识点的整理 + 背诵记忆（一般来说准备重点的知识点足够了）。此外，一定要仔细做一篇历年卷，有一定概率出现原题（比如我今年考的海森堡不确定性原理、还原论、呈展论都在历年卷中刚出现过）。

5. 考试建议：选择题不多，可以速通！物理类论述题写出概念定义即可。最后一部分大题其实很类似思政题目，可以按照思政考试来作答，写出自己的想法就行。建议可以多结合自己和自己的专业！
